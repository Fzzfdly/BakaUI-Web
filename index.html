v3.2.0 — The Progression Update

📅 January 30, 2026 🎮 Minecraft 1.21.132
This massive update redefines the server's economy and progression. We are introducing a "Play-to-Rank" questing ecosystem, a fully scripted Admin Event system, a high-stakes Casino, and a revamped Bounty system to enforce justice in the wilds.

🔰 Progression Overhaul

Progression
  • Play-to-Rank: Ranks are earned via Quest XP across a 12-tier ladder (Crew ➔ Exarch).
  • Dynamic Daily Quests: 5 unique daily quests reset at 12:00 AM (MYT) across varied archetypes.
  • Progressive Difficulty: Objectives dynamically scale up and yield better returns as you climb tiers.
  • Milestone Alerts: Automatic UI status warnings tracking progress at 25%, 50%, and 75%.

⚔️ Bounty System 2.0

Combat / Eco
  • Outlaw Tagging: Slaying an innocent player instantly labels you an Outlaw.
  • Automated Penalties: Triggers a 20% bank balance reduction penalty (min. $5,000 fine).
  • Combat Debuffs: Curses active outlaws with Weakness & Mining Fatigue for 30 seconds.
  • Global Broadcasts: Outlaw locations and actions broadcast coordinates across the entire server.
  • Live Menu Tracking: Bounty Menu tracks real-time dimension and location vectors.

🌟 Special Quests

Admin Tools
  • Global Events: Server-wide collaborative milestones now append to the peak of the Quest Menu.
  • Creator Framework: Staff can deploy interactive challenges using the custom ?adminquests engine.

🎰 Casino 2.0

Minigames
  • Slot Machine: Integrated a 3-reel system offering a 10x progressive jackpot prize tier.
  • Gacha Lucky Boxes: Randomized dynamic drop table crate structures tiered from Common up to Legendary.
  • Coinflip: Performance optimizations applied to standard double-or-nothing systems.

🛠️ Bug Fixes & Optimizations

Performance
  • Rank Core: Dropped legacy transactional rank components to clean execution pipelines.
  • Scoreboard Engine: Visual styling refinements alongside structural thread optimizations.
  • Land Safeguards: Cached lookup methods implemented to drastically lower real-time block query processing load.
  • Reward Regressions: Addressed a timing conflict where daily retention streaks dropped tracking arbitrarily.
